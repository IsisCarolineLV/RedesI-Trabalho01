## Descrição

O objetivo deste trabalho é **transmitir uma mensagem de um transmissor para um receptor, bit a bit**, simulando o funcionamento básico de um sistema de comunicação de dados.

A implementação representa o envio sequencial dos bits da mensagem pelo transmissor e a recepção desses bits pelo receptor, respeitando a ordem de transmissão.

---

## Para executar o trabalho

Tenha o **Java 8** instalado.

### Verificar se o Java está instalado corretamente:

```bash
java -version
javac -version
```

Deve aparecer algo semelhante a:

```text
java version "1.8.0_431"
```

---

## Como compilar e executar:

1. Acesse, pelo terminal, a pasta onde está o arquivo principal do projeto.

   (A pasta contém um arquivo `Principal.java`.)

2. Compile o código:

```bash
javac Principal.java
```

3. Execute o programa:

```bash
java Principal
```
